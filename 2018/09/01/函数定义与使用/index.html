####函数定义与使用

  1. 将函数视为“一等公民”

    • 什么是“一等公民”

      • 能赋值给另外有一个变量
      • 能作为另一个函数的入参
      • 能作为一个函数的返回值

    • 函数的定义

      • 实现一个特点功能的代码块

      • 使用 def 关键字

    • 函数定义应视 高类聚，低耦合 为终极原则

  2. 高阶函数的用法（filter、map以及它们的替代品）

    • filter – 过滤函数

      • 接收一个func,和一个list

      • 遍历list按照func的规则过滤数据，返回一个新的list

  • map – 映射函数

    • 参数接收一个func，和一个list

    • 对传入的list的每一个元素按照func映射，返回一个新的映射之后的list

  • 代替品

    • 推导式/生成式

  1. 位置参数、可变参数、关键字参数、命名关键字参数

    • 位置参数 – 调用函数时可按照位置传参

    • 关键字参数 – 使用key-value形式传值

  • 可变参数 – 不定参, 使用 *args 和 **kwargs代替不定长的位置参数和不定长的关键字参数,

    • *args是元组形式传入
    • **kwargs是以字典形式传入

  • 命名关键字参数

    • 定义需要一个*号作为分隔符，后面的参数表示只能传递该名字的参数

  1. 参数的元信息（代码可读性问题）

    • 能够描述函数的入参与出参,同时不会影响代码执行

  1. 匿名函数和内联函数的用法（lambda函数）

    • 定义匿名函数

      • 一行代码就可完成代码块
      • lambda x,y,z: 表达式

  1. 闭包和作用域问题（LEGB）- global / nonlocal

    • 变量的作用域，即寻找变量的顺序，当前作用域 》嵌套作用域 》全局作用域 》python内置作用域
    • 当函数中嵌套函数，那么里层函数才拥有嵌套作用。
    • 函数内声明嵌套作用域变量 –》关键字 nonlocal
    • 函数内声明全局变量 –》关键字 global

  2. 装饰器函数（使用装饰器和取消装饰器）

    • 装饰器 - 代理模式 - AOP(面向切面编程)

      横切关注功(与正常业务逻辑无关的功能)能应该从正常的业务逻辑中剥离
      在Python代码中可以使用装饰器来剥离横切关注功能
      在Web项目中可以使用中间件(拦截过滤器)来实现横切关注功能

    • 简单理解

      在不修改原有代码的情况下给代码附加新的功能

      实现计算一个函数运行时间的装饰器

    • 引入warps装饰器，是保证在装饰器中不修改原函数的任何信息

      如果不加，一些数据会被改变，如函数名

    • 装饰器的调用
